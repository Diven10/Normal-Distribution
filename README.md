# Gaussian Playground

Gaussian Playground is an interactive web application for visualizing and exploring the normal (Gaussian) distribution. Users can manipulate key parameters — mean (μ) and standard deviation (σ) — in real time and instantly see how the bell curve responds, making it an intuitive tool for building statistical intuition. The app also lets users calculate probabilities across custom intervals, overlay multiple distributions for comparison, and experiment with real-world datasets to see how well they fit a normal model.

Built with React and Vite, Gaussian Playground is designed for students, educators, and data enthusiasts who want a fast, hands-on environment to deepen their understanding of one of the most fundamental concepts in statistics and probability. Whether you're learning about the empirical rule, exploring z-scores, or just curious about how data tends to cluster, this playground gives you the tools to experiment freely and learn visually.

# React + Vite

This template provides a minimal setup to get React working in Vite with HMR and some ESLint rules.

Currently, two official plugins are available:

- [@vitejs/plugin-react](https://github.com/vitejs/vite-plugin-react/blob/main/packages/plugin-react) uses [Oxc](https://oxc.rs)
- [@vitejs/plugin-react-swc](https://github.com/vitejs/vite-plugin-react/blob/main/packages/plugin-react-swc) uses [SWC](https://swc.rs/)

## React Compiler

The React Compiler is not enabled on this template because of its impact on dev & build performances. To add it, see [this documentation](https://react.dev/learn/react-compiler/installation).
